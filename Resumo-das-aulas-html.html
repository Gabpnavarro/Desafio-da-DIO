Resumo das Aulas de HTML

  • História dos Computadores

    • Resumidamente: o primeiro computador criado foi nos EUA em 1946, conhecido por ENIC eles eram enormes, seu tamanho aproximado era de um quarto, 15 m x 2,5 m, e o seu tempo de resposta para as informações, cerca de 1 segundos para executar 4.500 calculos matemáticos.

    Curiosidade: nessa época foi criado o nome BUG, esse nome foi dado quando uma peça do computador queimava e naquele local enchia de insetos, vale ressaltar que é uma das teórias.

  • O que são Clients e Serves?

    • Resumidamente: o serves é aquele computador em uma rede com a função de oferecer algum tipo de serviço, como, controlar, gerenciar os recursos nessa rede. O computador do cliente, clients, é o que ocupa dos serviços oferecidos pelo computador servidor (serves).

    Exemplo do cotidiano: ao acessar uma página web, elas estão armazenados no servidor. Após digitar o endereço para o site desejado no seu computador (client), está pedindo a solicitação para acessar o site que se localiza no servidor, o servidor como respota te envia o acesso ao site web.
